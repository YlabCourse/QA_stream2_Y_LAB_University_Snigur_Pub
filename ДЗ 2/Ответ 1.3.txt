	ВОПРОС 3. Ответьте на вопросы:
	3.1. Приведите пример - ui-элемента из категории Input Controls.
	3.2. Является ли командная строка частью GUI?

	ОТВЕТ:
	3.1. Пример UI-элемента из категории Input Controls:
	Текстовое поле (Text Field). Текстовое поле позволяет пользователю вводить текстовую информацию 
(имя, адрес электронной почты, логтин, пароль и т.п.). 
Основные характеристики текстового поля:
	Место для ввода: Поле, в которое пользователь может кликнуть и начать вводить текст.
	Метка (Label): Обычно над полем или внутри него находится метка, указывающая, какую информацию нужно ввести 
(например, "Имя" или "Пароль").
	Подсказка (Placeholder): Внутри текстового поля может быть отображен текст-подсказка, который исчезает, когда пользователь 
начинает вводить данные.
	Кнопка подтверждения: Рядом с полем может располагаться кнопка для отправки введенной информации (например, "Отправить").
Текстовые поля могут иметь различные стили и типы (например, простое текстовое поле, поле для ввода пароля, поле для ввода 
электронной почты и т.д.), что позволяет адаптировать их под разные потребности интерфейса.

	3.2. Является ли командная строка частью GUI?
	Командная строка (CLI - Command Line Interface) не является частью GUI (Graphical User Interface).
GUI (графический пользовательский интерфейс) использует графические элементы, такие как кнопки, иконки, текстовые поля и меню, 
что позволяет пользователям взаимодействовать с системой с помощью мыши и других устройств.
CLI предоставляет текстовый интерфейс, где пользователи вводят команды с помощью клавиатуры для выполнения операций. 
Он не использует графические элементы и основывается исключительно на текстовом вводе и выводе.
	Командная строка и графический интерфейс пользователя представляют собой два разных подхода к взаимодействию с компьютером.
